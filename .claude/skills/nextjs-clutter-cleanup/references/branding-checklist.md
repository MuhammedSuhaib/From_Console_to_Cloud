# Branding Cleanup Checklist

## Hidden Branding Locations

1. **Tab Title:** Change "Create Next App" in `layout.tsx`.
2. **Meta Description:** Replace "Generated by create next app".
3. **Favicon:** Ensure you check both `/app` and `/public` directories.
4. **Social Links:** Check the default footer links in `page.tsx`.
5. **Console Logs:** Check for any default Next.js "Welcome" logs in root layouts.
6. **SVG Assets:** Look for `next.svg` and `vercel.svg` in public directory.
7. **CSS Classes:** Check for default Next.js class names that may contain branding.
8. **Comments:** Remove any default comments mentioning Next.js or Vercel.
9. **Links:** Remove any links to Next.js documentation or Vercel deployment pages.
10. **Images:** Check for any default placeholder images or logos.
11. **Manifest:** Verify `manifest.json` doesn't reference default assets.
12. **Apple Touch Icons:** Check for default apple touch icons in various sizes.

## Additional Cleanup Areas

- **package.json:** Remove any Next.js starter comments
- **README.md:** Replace default Next.js documentation with project-specific info
- **Documentation files:** Remove any default Next.js guides
- **Configuration files:** Check for any default Next.js configurations that may contain branding
