# AI Chatbot User Guide

This guide explains how to interact with the AI Chatbot in the Micro Task AI application, including available commands, conversation management, and best practices.

## Overview

The AI Chatbot is a conversational assistant that helps you manage your tasks using natural language commands. It's integrated into the dashboard and provides real-time responses with tool execution indicators.

## Accessing the Chatbot

### Opening the Chat Interface
- Click the **speech bubble icon** at the bottom-left of the screen
- The chat interface slides up from the bottom on mobile or appears as a sidebar on desktop

### Closing the Chat Interface
- Click the **X button** in the top-right corner of the chat window
- Or click the speech bubble icon again to toggle it closed

## Basic Interactions

### Starting a Conversation
- Simply type your message in the input field at the bottom
- Press **Enter** or click the **send arrow** to submit
- Example: "Hi" or "Hello, help me add a task"

### Natural Language Commands

#### Adding Tasks
- "Add a task to buy groceries"
- "Create a task called 'finish report'"
- "I need to schedule a meeting with John tomorrow"
- "Add task: wash the car this weekend"

#### Listing Tasks
- "Show me my tasks"
- "What tasks do I have?"
- "List my pending tasks"
- "Show completed tasks"

#### Completing Tasks
- "Mark task #1 as done"
- "Complete the grocery task"
- "I finished the report"
- "Set task 'wash the car' to completed"

#### Updating Tasks
- "Change the grocery task to include milk"
- "Update task #2 with a new description"
- "Edit the meeting task to be on Friday"

#### Deleting Tasks
- "Delete the old task"
- "Remove task #3"
- "Cancel the meeting task"

## Conversation Features

### Conversation Management
- **New Conversation**: Click the **+ button** in the chat header to start fresh
- **Browse Conversations**: Click the **clock/history button** to see past conversations
- **Switch Conversations**: Click any conversation in the sidebar to load it
- **Delete Conversations**: Click the **trash icon** next to a conversation to delete it

### Real-time Feedback
- **Typing Indicator**: Shows "Wait! I am typing..." when AI is processing
- **Tool Status**: Displays "Tool Call ðŸ› : [tool name]..." when executing actions
- **Immediate Response**: Buttons provide instant visual feedback when clicked

### Message History
- **Navigation**: Use left/right chevrons in the chat header to navigate through message history
- **Auto-scroll**: New messages appear at the bottom and scroll into view
- **Persistent Storage**: Conversations are saved between sessions

## Best Practices

### Effective Task Management
1. **Be Specific**: Use clear, descriptive language when adding tasks
   - Good: "Add task: buy milk, bread, and eggs from the grocery store"
   - Better: "Create a task to buy groceries including milk, bread, eggs, and fruits"

2. **Reference Tasks Clearly**: When updating or completing tasks, be specific
   - Good: "Complete the grocery shopping task"
   - Better: "Mark task #3 as completed" (if you know the task number)

3. **Group Related Tasks**: Consider organizing similar activities
   - "Add tasks: call dentist, schedule appointment, confirm availability"

### Conversation Tips
1. **Maintain Context**: The bot remembers conversation context within the same conversation
2. **Correct Mistakes**: If the bot misunderstands, clarify: "No, I meant..."
3. **Be Patient**: Allow time for tool execution, indicated by tool status messages

## Troubleshooting

### Common Issues

1. **Bot Doesn't Respond**
   - Check internet connection
   - Verify you're logged in
   - Try refreshing the page

2. **Commands Not Working**
   - Ensure using clear, specific language
   - Try rephrasing the request
   - Check if the bot misunderstood context

3. **Tool Execution Fails**
   - Look for error messages in the chat
   - Retry the command
   - Contact support if issues persist

4. **Slow Response Times**
   - Large conversations may take longer to process
   - AI processing speed depends on complexity
   - Be patient during tool execution

### Getting Help
- Use natural language to ask for help: "How do I add a task?"
- Refer to this guide for command examples
- Check the chat history for previous successful interactions

## Privacy and Security

### Data Protection
- Conversations are tied to your user account only
- Other users cannot access your conversations
- Task data is isolated per user

### Information Sharing
- Only share necessary information in conversations
- Avoid including sensitive personal details
- Remember that conversation data is stored in the database

## Advanced Features

### Multi-step Operations
- Chain commands in a single message: "Add a task to buy groceries, then list all my tasks"
- Reference previous tasks: "Update the last task I created"
- Combine operations: "Complete task #1 and add a new task"

### Context Awareness
- The bot remembers recent conversation context
- Previous tasks and discussions influence responses
- Start new conversations for unrelated topics

- Use numbers when referencing specific tasks: "Complete task #3"

- Ask for summaries: "What have I accomplished today?"
